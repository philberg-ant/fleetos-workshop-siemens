# Challenge 4 — reference solution

This is the finished `fleetos-toolkit` plugin a participant should end up
with after Step 4, plus example output artefacts.

```bash
claude plugin marketplace add .
claude plugin install fleetos-toolkit@fleetos-workshop
cd ../../4_team_scale/starter   # or any FleetOS checkout
claude                          # then /fleet-audit
```

`AUDIT.md` and `audit.json` are sample outputs from one run - yours will
differ.
